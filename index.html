Mis proyectos realizados

Encriptador de texto

Mi primer proyecto en el Programa ONE Oracle, consistia en crear un encriptador de texto que solo utilize letras minusculas sin caracteres especiales, debia ser capaz de encriptar y desencriptar los mensajes en la misma pagina. La regla que debia cumplir era:

  • La letra "e" es convertida para "enter"
  • La letra "i" es convertida para "imes"
  • La letra "a" es convertida para "ai"
  • La letra "o" es convertida para "ober"
  • La letra "u" es convertida para "ufat"

Para crearlo utilize Html 5, JavaScript y CSS

Juego del ahorcado

Mi segundo Challenge entregado en mi participacion del Programa ONE Oracle junto a Aluta Latam. En esta ocasion debiamos diseñar y programar un juego del ahorcado utilizando mayormente Canvas. El juego debia cumplir con los siguientes requisitos:

  • Debe funcionar solo con letras mayúsculas
  • No deben ser utilizadas letras con acentos ni caracteres especiales"
  • No debe ser posible escribir números dentro del juego
  • Las letras equivocadas deben aparecer en la pantalla
  • El juego debe tener un campo para agregar palabras

Para crearlo utilize Html 5, JavaScript y CSS
